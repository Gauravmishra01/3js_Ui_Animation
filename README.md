## ğŸ“¦ Installation

### Follow these steps to set up the project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Gauravmishra01/3js_Ui_Animation.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd 3js_Ui_Animation
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

### Usage

1. Open your browser and go to the link displayed in the terminal (e.g., `http://localhost:3000`).
2. Interact with the UI to see the animations in action.
