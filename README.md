# 3js UI Animation

3js UI Animation is a project that showcases interactive and dynamic animations using Three.js, a popular WebGL library. This project combines the power of Three.js with a modern development setup to create engaging user interfaces.

## Features

- **3D Animations**: Leverage Three.js to create real-time 3D animations.
- **Interactive UI**: Smooth integration of animations with a responsive user interface.
- **Modern Development Tools**: Built with React and Vite for optimal development and build speed.

## Technologies Used

- [Three.js](https://threejs.org/): JavaScript 3D library.
- [React](https://react.dev/): Frontend library for building user interfaces.
- [Vite](https://vitejs.dev/): Next-generation frontend tooling.
- [ESLint](https://eslint.org/): For maintaining code quality.

## Installation

Follow these steps to set up the project locally:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Gauravmishra01/3js_Ui_Animation.git
    ```
2. **Navigate to the project directory**:
    ```bash
    cd 3js_Ui_Animation
    ```
3. **Install dependencies**:
    ```bash
    npm install
    ```
4. **Run the development server**:
    ```bash
    npm run dev
    ```

## Usage

1. Open your browser and go to the link displayed in the terminal (e.g., `http://localhost:3000`).
2. Interact with the UI to see the animations in action.
